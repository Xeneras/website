Definiton List

HTML Terms

HTML
Hyper Text Markup Language - Language created specifically for creating websites. Its a GUI
HTML element
When an open tag and a close tag is together with something or a content inside. Its gives commands to the device so that it can execute it
HTML tag
Keyword or Terms assigned to a piece of information.
Angle Bracket
They contain things like HTML elements and HTML attributes.
HTML attribute
Attributes are assigned values using the assignment operator. The value is enclosed in quotation marks("")
HTML comment
Comments contain information for the webmaster to see, but not for the viewer of the web page. They can also be used to make the browser temporarily igno parts of a page
HTML object element
individual component of an HTML document or web page, once this has been parsed into the Document Object Model
HTML special character
Characters that cannot be written directly from the keyboard. Instead they need special characters to denote them.
